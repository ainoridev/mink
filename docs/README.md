## Cómo contribuir a la documentación

1. Actualiza el documento que corresponda al cambio (no dejes solo el código).
2. Para decisiones estructurales, añade un ADR nuevo (número secuencial).
3. Usa comentarios `<!-- RELLENAR -->` en plantillas hasta completarlas.
